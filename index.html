Adwaita Web CSS Showcase

This page demonstrates various HTML elements styled with Adwaita Web CSS. The library aims to provide styles that mimic the GTK Adwaita theme. Interactivity is provided by custom page JavaScript or from `toast.js` and `app-layout.js` if applicable. The `components.js` file, which might contain Web Component definitions or JS factories, was not found during the build.
